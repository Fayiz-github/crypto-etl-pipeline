# Crypto ETL Pipeline

## Overview
This project implements a complete ETL pipeline using Python.

## Steps
1. Extract data from CoinGecko API
2. Transform using pandas
3. Load into PostgreSQL

## Tech Stack
- Python
- pandas
- psycopg2
- PostgreSQL

## How to Run
pip install -r requirements.txt
python main.py

## Output
Data is stored in PostgreSQL table `crypto_data`
